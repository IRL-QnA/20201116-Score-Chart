## 문제

* 메뉴 5.성적차트 생성
* 메뉴 5번을 선택하면 두 과목 총점을 구하여 10점당 "*" 한 개씩 출력

```text
>>메인메뉴<<
==========
 1.성적입력
 2.성적출력
 3.성적삭제
 4.성적변경
 5.성적차트
 9.종    료
==========
메뉴 선택 :
```

> 5.성적차트 선택시:

```text
>>성적차트<<
이름 : 홍길동
총점 : ******************(180점)
-----------
이름 : 홍두께
총점 : *************(130점)
-----------
이름 : 장독대
총점 : *******************(190점)
-----------
```
